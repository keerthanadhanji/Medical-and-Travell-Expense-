# Medical and Travel Expenses Management System

# Project Overview
The Medical and Travel Expenses Management System is a web-based application developed using HTML, CSS, and JavaScript. It helps users record, manage, and track medical and travel expenses efficiently. The application provides a simple interface for entering expense details and viewing them in an organized format.

# Features
- Add medical expense details
- Add travel expense details
- Store expense information
- Display records in a table
- Calculate total expenses
- User-friendly and responsive interface

# Technologies Used
- HTML5
- CSS3
- JavaScript

# How to Run the Project
1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in any web browser.
4. Enter medical and travel expense details.
5. View and manage the expense records.
